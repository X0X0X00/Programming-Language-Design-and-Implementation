

# 计算器解析器测试

## 测试文件说明

这些测试用于验证一个简单编程语言的解析器和解释器功能。每个测试文件包含用特定语法编写的程序，用来测试解析器对不同语言特性的处理能力。

### 测试内容分析

**test1.txt** - 基础变量声明和赋值
- 测试整数变量声明、读取输入、赋值操作、输出功能
- 验证基本的变量操作和I/O指令

**test2.txt** - 条件语句 (if-else)
- 测试if-then-else条件分支结构
- 验证条件表达式求值和分支跳转

**test3.txt** - do-od循环结构
- 测试do-check-od循环语法
- 验证循环条件检查和循环体执行

**test4.txt** - 浮点数支持
- 测试实数(real)类型的变量声明和赋值
- 验证解析器对浮点数字面量的处理

**test5.txt** - 嵌套循环
- 测试多层嵌套的do-od循环
- 验证复杂的循环控制流和变量作用域

**test6.txt** - if-elsif-else条件链
- 测试多分支条件语句
- 验证elsif关键字和复杂条件逻辑

**test7.txt** - 简单循环与递减
- 测试基础循环结构配合递减操作
- 验证循环中的变量修改

**test9.txt** - 多重条件分支
- 测试复杂的if-elsif-else条件链
- 验证多个条件表达式的正确求值

**test14.txt** - 注释处理
- 测试行内注释和文件末尾无换行符的处理
- 验证词法分析器对注释的过滤能力

**test18.txt** - 表达式和空白符处理
- 测试多行表达式和各种空白符的处理
- 验证解析器对格式化不规范代码的容错性

**test20.txt** - 复杂控制流
- 测试循环内嵌条件语句的复合结构
- 验证复杂程序逻辑的正确执行

## 使用方法

~cs254/bin/calc_parse < tests/test1.txt
